# Dee HR Management System

Dee is a Laravel, Inertia, and Vue-based HR management web application. It helps organizations manage core HR operations from one dashboard.

## Features

- Employee management
- Departments and designations
- Attendance tracking
- Leave management
- Shift scheduling
- Recruitment: job openings, applicants, and onboarding
- Payroll and payslips
- Expense management
- HR messages and announcements
- Reports and analytics
- Roles and permissions
- HR chatbot audit tracking

## Tech Stack

- Laravel 12
- PHP 8.2+
- Inertia.js
- Vue 3
- Vuetify
- Vite
- MySQL or SQLite
- Spatie Laravel Permission
